# OTIMIZAÇÃO AVANÇADA DE INTERNET - WINDOWS

## 📌 Sobre o Projeto
Este script tem como objetivo otimizar a conectividade de rede em sistemas Windows, aplicando ajustes avançados em TCP/IP, limpeza de cache e redefinição de parâmetros de rede.

É voltado para melhoria de:
- Latência
- Estabilidade da conexão
- Desempenho em jogos e aplicações em tempo real

---

## ⚙️ Funcionalidades

### 🔄 Limpeza de Rede
- Limpeza de cache DNS
- Renovação de endereço IP
- Limpeza de tabela ARP
- Reset de Winsock
- Reset de pilha TCP/IP

### 🚀 Ajustes TCP Avançados
- Auto-tuning habilitado
- RSS (Receive Side Scaling)
- RSC (Receive Segment Coalescing)
- TCP Chimney Offload
- Controle de congestionamento (CTCP)
- Desativação de ECN
- Desativação de Timestamps
